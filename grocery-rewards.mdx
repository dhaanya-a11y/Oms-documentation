# Offer Management System (OMS) Documentation

## CVP Offer

# Grocery Rewards

This document explains how to configure the **Grocery Rewards** offer in the Offer Management System (OMS).

---

## 1. Label

**(Categorize offers by travel, food, beauty)**

- Create a new label or choose an existing label from the dropdown.

---

## 2. Banner Title

**(Banner title to be shown to the customer in PWA)**

The Banner Title is displayed on the **PWA Home Page** as the main heading of the offer banner. It is the first thing customers see while browsing available offers.

![Banner Title Home](./images/Banner%20Title%201.png)

### Banner Title Value

**Earn Rewards on Grocery Shopping**

![Banner Title Offer](./images/Banner%20Title%202.png)

---

## 3. Banner Description

**(Banner description to be shown to the customer in PWA)**

The Banner Description is displayed below the Banner Title on the **PWA Home Page**. It gives customers a short overview of the offer and encourages them to click and learn more.

![Banner Description Home](./images/Banner%20Description%202.png)

### Banner Description Value

**Earn 2 Reward Points for every ₹100 spent on eligible grocery purchases.**

![Banner Description Offer](./images/Banner%20Description%203.png)

---

## 4. Display Title

**(Title of the Offer to be shown to the customer in PWA)**

The Display Title is shown on the **Card Features** page after the customer selects the offer from the PWA Home Page. It serves as the main heading for the offer.

![Display Title Home](./images/Display%20Title%201.png)

### Display Title Value

**Grocery Rewards**

![Display Title Offer](./images/Display%20Title%202.png)

---

## 5. Display Description

**(Description of the Offer to be shown to the customer in PWA)**

The Display Description is displayed on the **Offer Details** page below the Display Title. It explains the benefits of the offer before the customer uses it.

![Display Description Home](./images/Display%20dis.png)

### Display Description Value

**Shop at eligible grocery stores and earn 2 Reward Points for every ₹100 spent.**

![Display Description Offer](./images/display%20dis%202.png)

---

## 6. Display Order

**(Prioritize display order)**

The Display Order determines the priority in which offers are displayed. Offers with a lower display order appear before offers with a higher display order.

## 9. How It Works

1. Shop at eligible grocery stores using your eligible card.
2. Spend on qualifying grocery purchases.
3. Earn **2 Reward Points** for every ₹100 spent on eligible transactions.
4. Reward Points will be credited according to the rewards program timeline.
5. Redeem your accumulated Reward Points through the rewards program.

---

## 10. Terms and Conditions

- Valid on eligible grocery purchases only.
- Earn 2 Reward Points for every ₹100 spent.
- Cancelled or refunded transactions are not eligible.
- Standard terms and conditions apply.
